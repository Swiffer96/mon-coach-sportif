Pour la prise de masse, une cible simple est souvent autour de 1,6 à 2,0 g/kg/jour.

Après la séance

Idées de repas simples

Repas solidePoulet ou œufs + riz/pâtes/pommes de terre + légumes + huile d'olive.
Collation rapideSkyr/fromage blanc + banane + flocons d'avoine + miel.
Soir tardifYaourt grec ou fromage blanc + fruit. Digeste, pas trop lourd.
Après jambes/salleVrai repas avec protéines + glucides. Évite de finir la journée trop bas en calories.

Douleur / prudence

Sauvegarde des données

Tes données restent sur ton appareil. La sauvegarde JSON est la copie de sécurité à garder dans iCloud Drive, Fichiers ou ton Mac.

Important : Safari/Chrome peuvent supprimer les données si tu vides l'historique ou les données de site. Sauvegarde régulièrement.
